CSS

Lenguaje usado para definir y crear la presentación de un documento estructurado escrito en HTML, XML2, XHTML HTML5

Boostrap

Framework para diseño de sitios y aplicaciones web. Contiene plantillas de diseño con tipografía, formularios, botones, cuadros, menús de navegación y otros componentes de diseño basado en HTML y CSS

Grunt

Librería JavaScript que nos permite configurar tareas automáticas y así ahorrarnos tiempo en nuestro desarrollo y despliegue de aplicaciones webs.

Yeoman

Conjunto de herramientas construidas sobre Node.js para simplificar las tareas repetitivas que están involucradas en la realización de cualquier proyecto.

Gestor de paquetes, instala y actualiza dependencias de terceros.

Angular.js

Framework para el desarrollo WEB, en lenguaje Javascript, diseño de patrones MVW

Impress.js

Librería para realizar presentaciones atractivas usa HTML, CSS3, transformaciones 3D y HTML5

JAVA

Java es un lenguaje de programación de propósito general, concurrente, orientado a objetos

HTML5

La más reciente version de HTML, Lenguaje de Marcado de Hipertexto

Javascript

Lenguaje de programación interpretado, utilizado principalmente para crear páginas web dinámicas, Front-End y mas recientemente Back-End mediante Node.js

Node.js

Entorno Javascript del lado del servidor, basado en eventos. Aprovechando el motor V8 de Google permite a Node proporcionarr un entorno de ejecución del lado del servidor que compila y ejecuta javascript a velocidades increíbles.

PhotoShop

Editor de gráficos rasterizados, usado principalmente de forma simple, para el retoque de fotografías y gráficos

Adobe Illustrator

Editor de gráficos vectoriales, destinado a la creación artística de dibujo y pintura para ilustración

Microsoft Office

Paquete de programas informáticos para oficina. (Procesador de Palabras, Hoja de Cálculo, Software de Presentaciones)

Microsoft Windows

Sistema Operativo desarrollado por Microsoft para computadoras casa y de negocios...

CouchDB

Base de datos de orientada a documentos, de código abierto, desarrollada en Erlang, un lenguaje de programación muy robusto y fiable, que se ejecuta sobre una máquina virtual de alto rendimiento en el área del paralelismo, es cross-platform. Las consultas utilizan un motor orientado a tablas mediante JavaScript.

MySQL

Sistema gestor de bases de datos (SGBD, DBMS por sus siglas en inglés) muy conocido y ampliamente usado por su simplicidad y notable rendimiento

Sublime Text

Editor de código multiplataforma, ligero. Es una herramienta concebida para programar sin distracciones. Plagado de plugins de utilidad asombrosa por su sencillez y simplicidad con el mas alto rendimiento y poder.

Android

Sistema operativo propiedad de google, disponible para dispositivos portátiles, como tablets y smartphones.

Jetty

Jetty es un servidor HTTP 100% y un contenedor de Servlets escrito en Java, Jetty se enfoca en crear un servidor web sencillo, eficiente, empotrable y pluggable. El tamaño tan pequeño de Jetty lo hace apropiado para ofrecer servicios Web en una aplicación Java empotrada.

Bitbucket

Servicio de alojamiento basado en web, para los proyectos que utilizan el sistema de control de revisiones Mercurial y Git.

ZK

Framework de aplicaciones web en AJAX, completamente escrito en Java permite una completa interfaz de usuario para aplicaciones web centrada en Servidor-Servidor. Altamente escalable, eficiente interfaz gráfica limpia y sencila usa patrones MVC, MVVM.

IntelliJ IDEA

Ambiente de desarrollo integrado (IDE) para el desarrollo de Aplicaciones Java

JetBrains WebStorm

IDE para Javascript desarrollado por JetBrains. Permite el desarrollo de capa cliente en JS pero también de la capa servidor con Node.js

Appcelerator

Plataforma de desarrollo de aplicaciones móviles, que incluye la posibilidad de integrar HTML5 y el servicio de alojamiento en la nube Appcelerator Cloud Services (ACS). Con esta herramienta desarrollo aplicaciones nativas para iPhone, Android.

Apache Felix

Implementación de código abierto de la especificación OSGi Versión 4

Equinox

Implementación certificada de OSGI versión 4.x, provee en tiempo de ejecución implementar una aplicación como un conjunto de "paquetes" que utilizan la infraestructura de servicios comunes.

OSGi

Open Services Gateway Initiative (OSGi). Su objetivo es definir las especificaciones abiertas de software que permita diseñar plataformas compatibles que puedan proporcionar múltiples servicios.

eXtreme Programming (XP)

Metodología de desarrollo de la ingeniería de software. Es el más destacado de los procesos ágiles de desarrollo de software.

Maven

Herramienta de software para la gestión y construcción de proyectos Java, de Sonatype. Es similar en funcionalidad a Apache Ant, pero tiene un modelo de configuración de construcción más simple, basado en un formato XML.

Lenguaje Unificado de Modelado (UML)

El lenguaje de modelado de sistemas de software más conocido y utilizado en la actualidad; está respaldado por el OMG (Object Management Group). Es un lenguaje gráfico para visualizar, especificar, construir y documentar un sistema.inluyendo aspectos conceptuales tales como procesos de negocio, funciones del sistema, y aspectos concretos como expresiones de lenguajes de programación, esquemas de bases de datos.

Apache Ant

Herramienta usada en programación para la realización de tareas mecánicas y repetitivas, normalmente durante la fase de compilación y construcción (build). Es un software para procesos de automatización de compilación, desarrollado en lenguaje Java

XML

eXtensible Markup Language, es un estándar para el intercambio de información estructurada entre diferentes plataformas. Se puede usar en bases de datos, editores de texto, hojas de cálculo y casi cualquier cosa imaginable.

XStream

Librería Java que permite serializar objetos Java a XML y realizar su posterior recuperación. Se caracteriza por su eficiencia y simplicidad de uso, generando documentos XML legibles.

JSON

JavaScript Object Notation, formato ligero para el intercambio de datos. JSON es un subconjunto de la notación literal de objetos de JavaScript que no requiere el uso de XML. La simplicidad de JSON ha dado lugar a la generalización de su uso, especialmente como alternativa a XML en AJAX.

JasperReport

Herramienta de diseño visual para JasperReports, libreria que nos permite crear informes-reportes, donde los informes se definen en un xml para posteriormente ser compilado por las librerías JasperReports (.jasper)

RESTful

Transferencia de Estado Representacional (Representational State Transfer) o REST es un estilo de arquitectura software para sistemas hipermedia distribuidos como la World Wide Web.

RestLet

Framework Restlet implementación del "API Restlet", una API neutral apoyar los conceptos de REST y facilitar el manejo de las llamadas para ambas aplicaciones del lado del cliente y del lado del servidor

NetBeans IDE

Entorno de desarrollo integrado, herramienta para escribir, compilar, depurar y ejecutar programas. Está escrito en Java, pero es usado para cualquier otro lenguaje de programación mediante extensiones plugins.

Jersey

Framework para desarrollo de servicios web RESTful en Java que proporciona soporte para JAX-RS APIs y sirve como JAX-RS (JSR 311 y JSR 339) Referencia de Aplicación.

JAX-RS

API del lenguaje de programación Java que proporciona soporte en la creación de servicios web de acuerdo con el ​​estilo arquitectónico Representational State Transfer (REST).

GlassFish

Servidor de aplicaciones de software libre desarrollado por Sun Microsystems, ahora Oracle Corporation, que implementa las tecnologías definidas en la plataforma Java EE y permite ejecutar aplicaciones que siguen esta especificación.

Corel Draw

Aplicación de diseño gráfico vectorial, está diseñada para suplir múltiples necesidades, como el dibujo, la maquetación de páginas para impresión y/o la publicación web, todas incluidas en un mismo programa.

TortoiseSVN

Cliente para el sistema de control de versiones Apache Subversion. Administra archivos y directorios a lo largo del tiempo. Los archivos se almacenan en un repositorio central.

Spring Security

Framework que gestiona todo lo relativo a la seguridad de una aplicación web, desde el protocolo de seguridad, hasta los roles que necesitan los usuarios para acceder a los diferentes recursos de la aplicación.

SEO

Consultor de negocios que puede ayudar a una empresa a aumentar su presencia en la web para que pueda llegar a más clientes potenciales. El especialista en SEO evaluará la posición actual de la empresa en internet, prestando especial atención a los resultados de búsqueda.

Diseño UX

“Diseño de Experiencia de Usuario” es una filosofía de diseño que tiene por objetivo la creación de productos que resuelvan necesidades concretas de sus usuarios finales, consiguiendo la mayor satisfacción y mejor experiencia de uso posible con el mínimo esfuerzo.

Jenkins

Jenkins es un servidor de integración continua, La base de Jenkins son las tareas las cuales automatiza para el desarrollo de un build, a Jenkins se le indica qué es lo que hay que hacer y este construye mediante el build.

OpenShift

Producto en la nube de plataforma como servicio (PaaS) de Red Hat. Este software funciona como un servicio que es de código abierto. Los desarrolladores pueden usar Git para desplegar sus aplicaciones Web en los diferentes lenguajes de la plataforma.
